## 技术实现

### 使用 Streamlit 原生组件

```python
# 创建5列布局：第一列占2份，其余各占1份
row1_cols = st.columns([2, 1, 1, 1, 1])

# 第一列：作者和时间
with row1_cols[0]:
    st.markdown(f"**👤 {author_name}**")
    st.markdown(f'📅 {create_time}')

# 第2-5列：统计数据（评论、点赞、分享、收藏）
for idx, (label, count, icon) in enumerate(stats):
    with row1_cols[idx + 1]:
        # 图标
        st.markdown(f'{icon}')
        # 数值
        st.markdown(f'**{count}**')
        # 标签
        st.markdown(f'{label}')
```

## 优势

1. ✅ **真正的横向布局**：使用 Streamlit 原生列组件
2. ✅ **可靠渲染**：不依赖复杂的HTML/CSS，避免显示原始代码
3. ✅ **响应式设计**：自动适配不同屏幕宽度
4. ✅ **易于维护**：代码简洁清晰
5. ✅ **视觉美观**：渐变背景 + 圆角边框 + 统一配色
